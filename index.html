Deep Learning for Anatomically-Consistent Airway Tree Modeling: Visual Features, Loss Functions, Learning Strategies, and a Novel Detection Task
Ali Keshavarzi
PhD Thesis, Télécom Paris – Institut Polytechnique de Paris, 2025

The thesis focuses on anatomically consistent airway tree modeling from chest CT scans using deep learning. It introduces convolutional sparse priors, improved loss functions, and reformulates airway segmentation as a structured detection problem to improve robustness and generalization across domains.

Boundary-Emphasized Weight Maps for Distal Airway Segmentation
Ali Keshavarzi, Elsa Angelini
arXiv preprint arXiv:2503.00242, 2025

This work introduces the Boundary-Emphasized Loss (BEL), a loss function designed to improve airway segmentation in lung CT scans by prioritizing boundary voxels. It addresses common issues such as leakage, breakage, and class imbalance, with a focus on distal airway branches. Evaluations on ATM22 and AIIB23 show improved topology preservation and segmentation accuracy.

BifDet: A 3D Bifurcation Detection Dataset for Airway-Tree Modeling
Ali Keshavarzi, Quentin Bouniot, Benjamin M. Smith, Elsa Angelini
arXiv preprint arXiv:2604.24999, 2026

This work introduces BifDet, the first publicly available dataset for 3D airway bifurcation detection in thoracic CT scans. The dataset includes annotated bifurcation bounding boxes derived from the ATM22 cohort and is designed to support detection-based learning approaches. Baseline experiments using RetinaNet and DETR demonstrate its utility for benchmarking airway-tree modeling methods.
